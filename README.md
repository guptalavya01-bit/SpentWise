# TrackExpense

TrackExpense is a comprehensive expense tracking application designed to help you manage your finances efficiently.

## Project Structure

This project is divided into two main parts:

- **Frontend**: A React application built with Vite.
- **Backend**: A Node.js/Express application.

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn

## Getting Started

### Backend Setup

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the backend server:
   ```bash
   npm start
   ```

### Frontend Setup

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Features

- Track expenses and income
- View detailed breakdown of your finances
- Intuitive user interface

## Technologies Used

- **Frontend**: React, Vite
- **Backend**: Node.js, Express, MongoDB
